Detailed Description

GPIO Helper Library API.

Author
Mark Greer mgree.nosp@m.r@an.nosp@m.imalc.nosp@m.reek.nosp@m..com

Set of routines to provide easy-to-use ways to perform common Linux(tm) GPIO operations.
